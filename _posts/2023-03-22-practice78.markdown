### 몫 구하기    
https://school.programmers.co.kr/learn/courses/30/lessons/120805    
    
#### 변수    
정수 num1, num2     
    
#### 제한사항    
0 < num1 ≤ 100    
0 < num2 ≤ 100    
    
#### 순서    
소수점 이하의 값을 취하지 않아서 내림값을 반환했다.    
1. Math.flooe(num1 / num2) 반환    
    
#### 코드    
```JavaScript
function solution(num1, num2) {
    return Math.floor(num1 / num2);
}
```
