### 두 수의 차    
https://school.programmers.co.kr/learn/courses/30/lessons/120803    
    
#### 변수    
-50,000 ≤ num1 ≤ 50,000    
-50,000 ≤ num2 ≤ 50,000    
    
#### 제한사항    
-50,000 ≤ num1 ≤ 50,000    
-50,000 ≤ num2 ≤ 50,000    
    
#### 순서    
1. num1 - num2 반환    
    
#### 코드    
```JavaScript
function solution(num1, num2) {
    return num1 - num2;
}
```
