### 두 수의 곱    
https://school.programmers.co.kr/learn/courses/30/lessons/120804    
    
#### 변수    
정수 num1, num2     
    
#### 제한사항    
0 ≤ num1 ≤ 100    
0 ≤ num2 ≤ 100    
    
#### 순서    
1. num1 * num2 반환    
    
#### 코드    
```JavaScript
function solution(num1, num2) {
    return num1 * num2;
}
```
