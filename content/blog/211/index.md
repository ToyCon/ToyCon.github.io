1. main
    - C/C++에서 다양한 함수가 시작될 수 있지만 프로그램은 항상 main 함수로부터 시작됨    
    - 함수는 반환 값이 없을 수도 있지만 main은 항상 0을 반환하는 것이 일반적임    
2. printf    
    - 특정한 문자 출력하는 함수. stdio.h에 포함되어 있음    
3. system
    - 함수. 명령 프롬프트를 실행한 이후 키보드 입력하기 전까지 대기하는 기능을 주로 수행
    - system("pause");
